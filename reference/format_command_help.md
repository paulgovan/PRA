# Format help text for a single command

Format help text for a single command

Format Help for a Single Command

## Usage

``` r
format_command_help(cmd_name, cmd)

format_command_help(cmd_name, cmd)
```

## Arguments

- cmd_name:

  Character command name (without /).

- cmd:

  Command definition from the registry.

## Value

A markdown-formatted help string.
