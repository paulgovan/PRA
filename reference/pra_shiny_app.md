# Create the PRA Shiny App Object

Create the PRA Shiny App Object

## Usage

``` r
pra_shiny_app(model = "llama3.2", rag = TRUE, embed_model = "nomic-embed-text")
```

## Arguments

- model:

  Ollama model name.

- rag:

  Whether to enable RAG.

- embed_model:

  Ollama embedding model.

## Value

A Shiny app object.
